Hinweis zur Speicherung, Verarbeitung und Uebermittlung personenbezogener Daten

Im Rahmen dieser Webseite werden Daten gespeichert, verarbeitet. 


Welche Daten werden uebermittelt?

Diese Webseite benoetigt einen Namen und eine gueltige Email-Adresse zur Kontoerstellung. Im Rahmen des Anmeldevorgangs wird ein Cookie auf deinem Computer gespeichert, der bis zum Beenden des Browser eine ID enthaelt, die dich eindeutig identifiziert. Dieses Cookie wird Sitzung-Cookie genannt und enthaelt keine weiteren Informationen, als die Zuordnung deines Anmeldevorgangs. Im Rahmen des Spiels erstellst und uebermittelst du JavaScript-Code der die Geschaeftslogik deines Restaurants enthalt.



Wer hat Einsicht in deine Dokumente und Daten? 

Der Name des Kontos wird oeffentlich auf dieser Webseite gezeigt. Die Email-Adresse kann nur vom Administrator/Inhaber (Oliver Zimpasser) dieser Seite eingesehen werden. Der von dir erstellte JavaScript Programmcode kann immer von Administratoren und Inhabern dieser Seite gesehen werden, falls du es moechtest hast du allerdings auch die Moeglichkeit deinen Programmcode oeffentlich zu machen.

 

An wen werden deine Daten weitergegeben?

Deine Daten werden innerhalb dieser Webseite gespeichert. An Dritte erfolgt keine Weitergabe. 



Wie lange erfolgt die Speicherung?

Deine Daten werden auf diesem Server solange gespeichert, wie dein Konto hier am Spiel teilnimmt.
